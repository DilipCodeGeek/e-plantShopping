# e-plantShopping

## Paradise Nursery Shopping Application

The **e-plantShopping** repository contains the final project implementation of the Paradise Nursery Shopping Application built using React and Redux.

### Project Overview

Paradise Nursery is an online plant shopping application that allows users to:

- View a landing page with company information
- Browse different categories of plants such as Aromatic Plants and Medicinal Plants
- Add plants to a shopping cart
- Increase or decrease the quantity of items in the cart
- Remove items from the cart
- View the total number of items and total cost dynamically

### Technologies Used

- React
- Redux Toolkit
- React Router
- Vite

This repository (e-plantShopping) demonstrates state management using Redux and dynamic UI updates in a React application.
